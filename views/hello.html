Let's learn Twig

1. Rendering variables

Hello, {{ name }}! You are {{ age }} years old.

2. Conditional rendering

{% if age > 18 %}

You are above 18, so you can legally drive.

{% else %}

You are not above 18, so you cannot legally drive.

{% endif %}

3. Rendering a list from data

    {# If you know JavaScript, this syntax should be familiar #} {% for user in users %}
  • {{ user.name }}, {{ user.age }} years old
    • {% endfor %}

4. Rendering a table from data

{% set counter = 0 %} {% for user in users %} {% set counter = counter + 1 %} {% endfor %}
# Name Age Role
{{ counter }} {{ user.name }} {{ user.age }} {{ user.role }}
